**Project Title:** GCMS (Gigaclear Compensation Management System) - HR Onboarding Platform

**Type:** Professional

**Short Description:**
The GCMS is a comprehensive HR onboarding and compensation management system built for Gigaclear. It features a multi-step candidate starter form that guides new employees through onboarding process, including personal details, National Insurance information, health declarations, and other essential HR documentation. The system provides separate interfaces for candidates and HR staff, with robust authentication, form validation, and granular access control across 7 different user roles.

**My Role & Contribution:**
I was primary frontend developer responsible for building and maintaining the candidate onboarding experience and developed critical automation solutions. I developed the complete multi-step starter form system with 8+ steps including personal details, National Insurance, health declarations, and emergency contacts. I implemented comprehensive form validation using Formik and Yup, created reusable modal components for corrections and confirmations, and built progress tracking system. Most significantly, I engineered a sophisticated JavaScript automation script that solved a critical database authorization challenge - the system required manual database updates for each new input/output across 5 tables with 10,000+ rows total for 7 different roles. My AccessRights Query Generator automatically generates complete SQL scripts with 388+ UI items mapped to 67+ groups, handling 3 privilege levels (view, edit, hide) across 5 status levels, reducing what was hours of manual work to adding single configuration rows when new fields are added.
