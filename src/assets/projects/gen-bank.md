# Gen Bank Landing Page

This project is a high-performance landing page for a fictional digital bank. It focuses on **clean UI**, **responsive design**, and **fast loading times**.

## Features

- **Responsive Design**: Works on mobile, tablet, and desktop.
- **Glassmorphism**: Subtle effects for a premium feel.
- **Animations**: Using Framer Motion (or CSS transitions-micro animations).

## Tech Stack

- **React**: For the UI logic.
- **Tailwind CSS**: For rapid styling and utility-first approach.
- **Vite**: For the build tool.

You can view the source code on [GitLab](https://gitlab.com/Angelo_E_DZ/coding/react-js/landing-page).
