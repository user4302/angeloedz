**Project Title:** iTelaSoft Builder CMS - Enterprise Website Platform

**Type:** Professional

**Short Description:**
This is a Next.js-based enterprise content management system built with Builder.io that powers iTelaSoft's corporate website. The platform serves as a comprehensive digital hub showcasing the company's IoT, FinTech, mobile app development, and cloud engineering services. It integrates multiple form systems, marketing automation tools, and dynamic content management capabilities to support lead generation and customer engagement.

**My Role & Contribution:**
I made 89 commits to this project, primarily focusing on form system development and integration. My key contributions include:

- **WSO2 Campaign Integration:** Developed and implemented the complete WSO2 campaign contact form system, including API integration with Pardot and Salesforce endpoints
- **Form Architecture Refactoring:** Migrated form submission data management from localStorage to React Context for better state management and user experience
- **Form Validation & UX:** Implemented comprehensive form validation, including whitespace validation, proper error handling, and improved user feedback mechanisms
- **API Development:** Created and maintained multiple API routes including submitToPardot, auditForm, contactForm, and submitToSalesforce endpoints
- **Component Development:** Built reusable form components including CampaignContactv2, ConsultationForm, and various UI components
- **Bug Fixes & Optimization:** Resolved numerous form submission issues, validation problems, and improved overall form reliability
- **Code Quality:** Maintained code standards through linting fixes and proper TypeScript implementations
