# Vue Task Tracker

Stay organized with this lightweight task tracker. This project demonstrates **Vue 3 Composition API** usage and state management.

## Key Features

- **Add/Delete Tasks**: Simple CRUD operations.
- **Persistence**: Saves tasks to local storage.
- **Filter**: Filter by completion status.

## Why Vue?

Vue's reactive system makes it perfect for interactive tools like this.

Check the live demo [here](https://user4302-task-tracker.netlify.app/).
