# Security Policy

## Supported Versions

Currently, only the latest version of this project is supported.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it immediately by opening an issue on GitLab:

https://gitlab.com/Angelo_E_DZ/coding/vue-js/angeloedz/-/issues

Please include:
- A detailed description of the vulnerability
- Steps to reproduce the issue
- Potential impact assessment
- Any suggested fixes (if available)

## Response Time

We will respond to security reports within 48 hours and provide a timeline for addressing the issue.

## Security Best Practices

- Keep dependencies updated
- Use environment variables for sensitive data
- Implement proper input validation
- Use HTTPS in production
- Regular security audits

## Disclaimer

This project is a personal portfolio website and does not handle sensitive user data. However, we take security seriously and will address any reported vulnerabilities promptly.
